Detailed Cache Clearing Instructions:
Chrome / Edge
  1. Press Ctrl + Shift + Delete
  2. Select "Cached images and files"
  3. Click "Clear data"
  4. Refresh page with Ctrl + F5
Firefox
  1. Press Ctrl + Shift + Delete
  2. Select "Cache"
  3. Click "Clear"
  4. Refresh page with Ctrl + F5
Mobile
  1. Open browser settings
  2. Find "Privacy" or "Storage"
  3. Clear "Cached data" or "Site data"
  4. Refresh the page
